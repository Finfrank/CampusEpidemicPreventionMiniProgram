## 本项目具备以下特点：  
1.绑定个人信息，填写报表时无需再次填写  
2.区分教师、学生、辅导员、院长四类用户  
3.导出Excel表格，不同颜色标注发热人员   
  
## 依赖  
1.本项目使用云开发，无需域名，无需部署服务器  
2.导出Excel模块依赖exceljs库
